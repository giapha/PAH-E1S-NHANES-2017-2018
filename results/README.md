# Results

Generated aggregate result tables and session information are written here. The tracked CSV files are participant-safe summaries that reproduce the manuscript's journal-facing estimates.

Primary outputs include the six-biomarker E1S discovery screen, endocrine-selectivity and reservoir-ratio models, hepatic and lipid contrasts, LOD/BMI sensitivities, independent-cycle active-hormone comparisons, and survey-weighted spline diagnostics. Participant-level intermediate RDS files remain excluded from version control.
